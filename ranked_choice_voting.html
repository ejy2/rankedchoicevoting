How it works

  1. In Google Sheets, go to File → Download → CSV and drop the file above.
  2. The app auto-detects columns that look like "[Election] [Rank N]" (the default Google Forms format).
  3. Select which elections to tabulate, then hit Run Elections.
  4. Watch the animated Instant Runoff play out round by round — the lowest-vote candidate is eliminated each round until someone crosses 50%.
  5. Save this HTML file to your Google Drive — it works completely offline, no server needed.
